~~~~~~~~Readme~~~~~~~~

1. Contenu
(1) P5_01_dendrogramme.png >> fichier image du dendrogramme
(2) P5_02_code_nettoyage_HTML.nb >> fichier html du code de nettoyage
(3) P5_O2_code_nettoyage.rmd  >> fichier R du code de nettoyage
(4) P5_03_code_analyse_HTML.nb  >> fichier html du code d'analyse
(5) P5_03_code_analyse.rmd >> fichier R du code d'analyse
(6) P5_04_pays_clusters.csv >> fichier csv des pays et cluster d'appartenance
(7) P5_05_centroides.csv >> fichier csv des centroides des groupes
(8) P5_06_coord_centroides.csv >> fichier csv des coordonnées des centroides 
(9) P5_07_presentation.pptx >> fichier pptx de la présentation
(10) IN >> dossier contenant les données sources
(11) OUT >> dossier contenant les données produites
(12) PLOTS >> dossier contenant l'ensemble des figures produites au format png

2. Exécution
Dans R studio;
Pour exécuter le code R de nettoyage, ouvrir le fichier P5_02_code_nettoyage.rmd et exécuter l'ensemble des blocs
Pour exécuter le code R d'analyse, ouvrir le fichier P5_03_code_analyse.rmd et exécuter l'ensemble des blocs

En cas de problème de lecture ou d'ouverture des fichiers csv contenant les données, changer le "working directory" à la racine du dossier, par exemple : "C:\Users\maxim\Desktop\Formation Data Analyst\Projet  5.1\P5_VAYNE_Maxime"
